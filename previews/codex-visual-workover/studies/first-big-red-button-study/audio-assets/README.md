# Audio Assets

This folder is organized around the final deliverables for the first Big Red Button VR study and the reusable assets used to build them.

## Final deliverables

- `final/first-big-red-button-vr-study-instructions-final.mp3`
- `final/first-big-red-button-vr-study-demo-button-pressing-behavior-in-virtual-reality.mp4`
- `final/first-big-red-button-vr-study-demo-second-condition-button-pressing-behavior-in-virtual-reality.mp4`

## Reusable assets

- `reusable/audio/source/` - instruction takes and source music
- `reusable/audio/derived/` - extracted and intermediate helper audio
- `reusable/video/` - source video inputs
- `reusable/graphics/` - SVG artwork
- `reusable/models/` - 3D model source files
- `reusable/audio/local-only/` - oversized local assets kept out of GitHub

## Naming convention

Files use lowercase kebab-case names with explicit roles such as `source`, `derived`, and `final`.

## GitHub note

The raw `reusable/audio/local-only/accounting-plus-full-source.mp3` asset is ignored in Git because it exceeds GitHub's standard file size limit.
