## Designers Republic Reference Set

Collected on March 14, 2026 for the `codex/visual-workover` branch.

Purpose:
- Internal visual reference for BRB site styling exploration.
- Focused on poster systems, compressed typography, labeling, flat color blocking, and technical-grid treatments associated with The Designers Republic and related work.
- Not intended for redistribution; retain source attribution and rights context with the original publishers.

Observed motifs:
- Condensed grotesk and mono type used as hierarchy, not decoration.
- Hard-edged panels, bars, warning labels, and registration-mark style framing.
- Sparse palettes: off-white, black, signal red, occasional pale blue.
- Diagrammatic density: microcopy, codes, badges, and catalog-like structure.
- Flat image treatment instead of glass, blur, pill buttons, or soft gradients.
